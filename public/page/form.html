表单的作用和组成

表单的作用:	
       1.提供可视化的输入控件
       2.收集用户输入的信息，并提交请求给服务器
总结
       form自带提交请求的功能
       ajax提交不需要form支持

表单的组成:
     1.前端部分
            提供表单控件，与用户交互的可视化控件
     2.后端部分
            后台接口对提交的数据进行处理




表单form

  <form action="" method="" enctype=""></form>
    属性：
        1. action="" 定义表单提交时发送的动作(接口url)    
		            如果没有值，默认提交给本页
        2. method="" 定义表单提交方式
                取值：
                       1.get 默认值
                         特点：明文提交，提交的内容会显示在浏览器地址栏上(不安全)
                         提交的数据有大小限制，最大为2kb
                         向服务器要数据时，使用get方法
                       2.post
                         特点：隐式提交，提交的数据不会在地址栏显示
                         提交数据没有大小限制
                         要传递数据给服务器的时候，使用post
                       3.其它方法
                          delete    put    option

        3. enctype=""
                指定表单数据的编码方式
                允许将什么样的数据提交给服务器
                取值：
	      1.application/x-www-form-urlencoded  默认值，允许将任意字符提交给服务器(文件除外)
              2.text/plain 允许提交普通字符给服务器
              3.multipart/form-data 允许提交文件给服务器

input元素

     <input type="" name="" value="" disabled>
         在页面中，提供了各式各样的输入控件
         比如，文本框，密码，单选按钮，多选按钮等等
         属性：
            type：    指定input元素的类型
            name：  为控件定义名称，提供给服务器端使用（name必须写）
            value：   控件的值，传递给服务器的值
                           例外，对于按钮来说，value是设置按钮显示的内容
            disabled：禁用，不能修改值，也不能提交
                          无值属性，在标签使用的时候，不写值
                          后期使用js修改值，认为该值为true

    4. 隐藏域

         隐藏域：
         



    隐藏域
         type="hidden"
                想把数据提交给服务器，但是不想展示给用户看
                这种数据，放到隐藏域中
                用户看不见，却又可以提交




    5.文件选择框

  



    文件选择框（上传文件使用）
         type="file"
             注意：使用文件选择框有两个前提
                     1.必须使用post方法提交
                     2.需要把form的enctype属性，设置为multipart/form-data

             属性：multiple 无值属性，选择多个文件

  下拉选择

  
	
      



    下拉选
        <select>
          <option></option>
          <option></option>
          ......
        </select>
    注意：
          1.如果option没有value属性，select的value等于被选中的option的内容
          2.如果option有value值，select的value值是选中option的value
    属性:
        select:  
                 name 
                 value
                  size       默认值为1,1的时候是下拉选择框
                               >1的时候，是滚动选择框
              multiple    无值属性，设置多选
        option:
                  value     定义选项的值
               selected   无值属性，默认选中

     新表单元素
     在html5版本中，新提出来的表单控件
    1.邮箱，提交的时候验证@及@后面有一个字节
         <input type="email" name="email">
    2搜索类型，自带快速清除功能
         <input type="search" name="search">
    3. url类型验证，提交的时候验证有http://开头
         <input type="url" name="url">
    4.电话号码，移动端使用时，弹出虚拟键盘
         <input type="tel" name="tel">
    5.数字类型
         <input type="number" max="10" min="5" step="2">
           max="10" 能接受的最大值
           min="5"  能接受的最小值
           step="2"  步长 
    6.范围类型
         <input type="range">
            max="20" 
            min="16" 
            step="2"  同上
    7.颜色类型
         <input type="color">
    8.日期类型，提供了一个日期选择控件
         <input type="date">
    9.月份类型，提供了一个月份选择控件
         <input type="month">
   10.周类型
         <input type="week">
